# OTServer Custom

Projeto pessoal de customização de OTServer com alterações em C++, Lua e integração futura com banco de dados.

Atualmente estou desenvolvendo um sistema mais avançado de controle de summons, com foco em mecânicas de gameplay e integração com banco de dados.

### Sistema de Summon (em evolução)

- Estruturação de dados para controle de estado do summon
- Persistência de informações via banco de dados
- Planejamento de progressão de habilidades do summon
- Associação entre jogador (owner) e summon

### Funcionalidades planejadas

- Sistema de evolução/progressão do summon
- Habilidades baseadas em nível e uso (spell system)
- Controle direto do summon pelo jogador (movimentação via input)
- Sistema de estados (ativo, controlado, automático)

### Objetivo técnico

- Melhorar integração entre client e servidor
- Trabalhar com persistência de dados (database)
- Criar sistemas escaláveis dentro da arquitetura do OTServer
