# OTServer Custom

Projeto pessoal focado em desenvolvimento e customização de OTServer, com implementação de sistemas e correção de bugs.

## Tecnologias
- C++
- Lua
- Git

## Experiência prática
- Análise da estrutura interna do servidor OTServer
- Desenvolvimento de scripts em Lua
- Correção de bugs em sistemas existentes
- Testes e validação de funcionalidades no jogo

## Funcionalidades implementadas
- Alteração na lógica de target dos monstros para priorizar summons
- Modificação direta no core do servidor
- Impacto no comportamento de combate

### Controle de summon pelo jogador (Client + Server)
- Desenvolvimento de sistema para movimentar summon via teclado (setas)
- Integração entre client e servidor para envio de comandos
- Sincronização de movimentação em tempo real

### Sistema de controle do personagem (Owner Lock)
- Implementação de lógica para travar o personagem principal durante controle do summon
- Gerenciamento de estados do player (ativo/controlando summon)

### Customizações no Client
- Modificações na interface e input do cliente
- Ajustes para suportar novas mecânicas de controle

## Aprendizados
- Integração entre client e server
- Lógica de programação aplicada em ambiente real
- Organização de código em projetos maiores
